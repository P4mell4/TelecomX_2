# 📊 Telecom X – 2 : Previsão de Evasão de Clientes (Churn)

## 📌 Sobre o Projeto

Este projeto tem como objetivo desenvolver modelos de **Machine Learning** capazes de prever a evasão de clientes (churn) na empresa fictícia **Telecom X**.

A análise foi realizada a partir de um conjunto de dados previamente tratado na primeira etapa do desafio, contendo informações sobre perfil de clientes, serviços contratados e histórico de cobrança.

A partir desses dados, foram aplicadas técnicas de **preparação de dados, análise exploratória, modelagem preditiva e interpretação dos resultados**, com o objetivo de identificar os principais fatores que influenciam o cancelamento de serviços.

---

# 🎯 Objetivos

- Preparar os dados para modelagem preditiva
- Analisar a correlação entre variáveis
- Construir modelos de classificação para prever churn
- Avaliar o desempenho dos modelos
- Identificar os fatores que mais influenciam a evasão de clientes
- Propor estratégias de retenção baseadas nos resultados
